Operating System.

*Operating System is a software application that controls the software and hardware part of the computer.

*Operating systems are categoriesed according to types depending on how they work,

Types Of Operating System Include;
    • 1.Batch operating system.

    -User prepares job offline and submit to computer operator.

      2.Real-time operating system.

    -Time interval to process and receive respond input is very small.

      3.Distributed operating system.

    -Uses many processors located in different machines to provide very fast computation yo users.

      4.Network operating system.

    Provides capability to server to manage data user group and other network functions.

      5.Mobile operating system.

    Designed to power smartphones ,tablets and wearable devices.
